Fun Learn is a rfuvsyionsl game written in Python 3 using HTML, CSS, Javascript and the Flask Framework to bring it all together.

The point of this game is to guess the answer to a question presented on screen and if guessed right, points will be added to your total score. If your score is high enough, you will get put onto the highscore board.

It helps young students learn at home during COVID-19 in a competetive and fun fassion by being able to compete with their friends.
